# Fixture Layout Upgrade

This version changes the fixture card layout to:

Team A    vs    Team B
          date, time

## What changed
- Team names are displayed on one row
- VS is smaller and centered
- Date/time is centered below VS
- The group letter is removed from each match row because fixtures are already separated by Group A, Group B, etc.

## Setup
1. Upload all files to your GitHub repository root.
2. Commit changes.
3. Wait 1-3 minutes.
4. Open `fixtures.html`.
5. Press Ctrl + F5.
