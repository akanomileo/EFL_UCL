# Results Same Layout Upgrade

This version makes Results match the fixture layout.

## Results format
Team A      2 - 1      Team B
            date, time

## Also included
- Results are separated by Group A, Group B, etc.
- Fixtures keep the same Team A vs Team B layout.
- Score appears in the middle for completed matches.
- VS appears in the middle for upcoming fixtures.

## Setup
1. Upload all files to GitHub repository root.
2. Commit changes.
3. Wait 1-3 minutes.
4. Open results.html.
5. Press Ctrl + F5.
