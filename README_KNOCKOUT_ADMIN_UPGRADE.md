# UCL Knockout Admin Upgrade

This version adds an admin area for knockout matches after the group stage.

## New admin tab
Admin → Knockout Fixtures

You can:
- View qualified teams from group standings
- Generate first knockout round
- Generate next knockout round after results are entered
- Set knockout fixture date/time
- Set deadline per knockout round
- Enter knockout results
- Clear knockout fixtures without deleting group stage

## How it works
1. Finish group stage results.
2. Open Admin → Knockout Fixtures.
3. Click Generate First Knockout Round.
4. Set dates/times and deadlines.
5. Enter knockout results.
6. Click Generate Next Round.
7. Repeat until Final.

## Important
Automatic knockout generation works best when the qualified team count is 2, 4, 8, 16, or 32.

If a knockout match is a draw, the system cannot choose a winner automatically. Admin must edit the result to a winning score first.
