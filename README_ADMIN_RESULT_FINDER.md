# Admin Result Finder Upgrade

The League Results and Knockout Results sections now include:

- Instant search by team or fixture name
- Team dropdown filter
- Round filter
- Pending/completed filter
- Pending-only default view when pending matches exist
- Clearer team names with team logos
- Live visible-fixture count
- Reset button

The existing Save League Results and Save Knockout Results workflows are unchanged. Hidden filtered rows remain in the form and are not deleted.
