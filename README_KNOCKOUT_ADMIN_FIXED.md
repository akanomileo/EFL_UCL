# UCL Knockout Admin Fixed Version

This fixed version definitely adds:

Admin → Knockout Fixtures

Use it after group stage to:
- Generate first knockout round
- Generate next knockout round
- Set knockout deadlines
- Set knockout date/time
- Enter knockout results

Important setup:
1. Upload all files to the GitHub repository root.
2. Commit changes.
3. Wait 1-3 minutes.
4. Open /admin.html.
5. Press Ctrl + F5.
6. You should see "Knockout Fixtures" in the admin left menu.

If you still do not see it, open GitHub app.js and search for:
Knockout Fixtures
