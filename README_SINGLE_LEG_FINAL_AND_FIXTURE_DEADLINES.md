# Single-Leg Final + Fixture Deadline Update

## Changes

1. Knockout Playoff, Round of 16, Quarter Finals, and Semi Finals remain two-legged.
2. The Final is generated as one single match only.
3. Older saved two-leg Finals are automatically consolidated into one Final fixture.
4. The public Fixtures page shows the applicable deadline below every fixture:
   - League Phase fixtures use the League Result Deadline.
   - Knockout fixtures use the deadline configured for their knockout round.
5. Passed deadlines use a red deadline label.
6. Admin and public format descriptions were updated to reflect the single-leg Final.

## Admin setup

- Set the League Phase deadline in **Admin → League Results**.
- Set knockout deadlines in **Admin → Knockout → Knockout deadlines**.
- Set fixture dates and times in **Admin → Fixtures** or **Admin → Knockout**.
