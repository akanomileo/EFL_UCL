# Standings Number Alignment Fix

This version fixes standings table alignment.

## What changed
- P, W, D, L, GF, GA, GD, Pts numbers are centered.
- Team names stay left-aligned.
- Numeric columns have consistent width.
- Mobile table spacing is improved.

## Setup
1. Upload all files to your GitHub repository root.
2. Commit changes.
3. Wait 1-3 minutes.
4. Open standings.html.
5. Press Ctrl + F5.
