# Tournament Name Admin Upgrade

This version adds an admin field for changing the tournament name.

## What changed
- Removed the text: "UCL Format Tournament"
- Added editable Tournament Name in Admin → Tournament Settings
- The website title, header brand, homepage title, and footer now use your saved tournament name
- Existing Supabase data still works

## Setup
1. Upload all files to GitHub and commit changes.
2. Open your website `/admin.html`.
3. Login with your admin PIN.
4. Go to Tournament Settings.
5. Edit Tournament name.
6. Click Save Settings.
7. Refresh the website.

## Existing Supabase users
You do not need to recreate your database. Just save Tournament Settings once from the admin page.
