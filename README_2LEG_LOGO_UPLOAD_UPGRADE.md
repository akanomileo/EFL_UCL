# UCL 2 Leg + Team Logo Upload Upgrade

Changes included:
- Removed the home page sentence: "24-team UCL format: League Phase → Playoff → Round of 16 → Final".
- Changed home page knockout label from "Best of 2" to "2 Leg".
- Updated admin/bracket wording from Best of 2 to 2 Leg where visible.
- Added team logo upload in Admin → Teams + Logo Upload.
- Logos display on Teams page, Standings table, Top Scorers page, and Home top scoring section.
- Uploaded logos are resized to 180x180 and saved in browser localStorage with team data.

How to use:
1. Open admin.
2. Go to Teams + Logo Upload.
3. Create/add teams.
4. Upload logo for each team in the Team List table.
5. Check Top Scorers / Teams / Standings pages.
